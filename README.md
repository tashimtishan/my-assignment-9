docappoint

live site: https://my-assignment-9.vercel.app

a full stack doctor appointment booking system where patients in dhaka can find doctors and book appointments online.

 browse all available doctors with their specialty, experience, hospital, location, consultation fee and availability schedule
 book appointments instantly by filling in patient details, preferred date, time and reason for visit with a confirmation toast on success
 secure login and registration system using better auth with google oauth support and jwt protected backend routes
 personal dashboard where logged in users can view all their bookings, update appointment details or cancel bookings anytime
 update profile feature lets users change their name and profile photo directly from the dashboard without leaving the page
 fully responsive design that works smoothly on mobile, tablet and desktop screens
